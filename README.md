# Fake-Profile-Detection-System

This project implements an AI-powered Fake Profile Detection System that identifies whether a social media profile is real or fake using Machine Learning techniques.

The system analyzes profile characteristics such as followers, following count, posts, bio length, and account age to classify profiles and provide prediction confidence scores through an interactive Streamlit web application.

## Project Overview

With the rapid growth of social media platforms, fake profiles have become a major concern. These accounts are often used for scams, misinformation, spam, and fraudulent activities.

This project uses a Random Forest Classifier to automatically detect fake profiles based on profile-related attributes, providing a scalable and efficient alternative to manual verification methods.

## Results

The model successfully classifies social media profiles as real or fake using profile-related attributes.

Key observations:

- Profiles with higher engagement and longer account age are generally classified as genuine.
- Profiles with low activity and suspicious behavior patterns are more likely to be classified as fake.
- The system provides prediction confidence scores for better interpretability.
